# solana-mev

A collection of tools and libraries for MEV research and development on the Solana blockchain.

## Watcher
`./watcher` includes a Golang application for monitoring Solana transactions, Jito bundles and extracting sandwich opportunities.

## Analyst
`./analyst` includes scripts for analyzing and visualizing MEV data

## DB
`./db` includes scripts to set up and manage a Clickhouse database for storing data
